## 📝 Descripción del Proyecto
**Nexus Systems** es una suite de herramientas analíticas diseñada para optimizar la toma de decisiones en entornos de ingeniería. La plataforma integra diversos módulos funcionales que demuestran el dominio de estructuras de datos, lógica algorítmica y programación orientada a objetos (POO) en Python.

---

## 🛠️ Tecnologías Utilizadas
*   **Lenguaje:** Python 3.x
*   **Framework:** Streamlit (Interfaz de usuario)
*   **Librerías de Datos:** Pandas & NumPy
*   **Despliegue:** Streamlit Cloud

---

## 🚀 Módulos del Sistema
1.  **Home:** Ficha técnica del desarrollador y descripción del núcleo del proyecto.
2.  **Ejercicio 1 - Flujo de Caja:** Gestión financiera dinámica utilizando **listas** de Python para el control de ingresos y egresos.
3.  **Ejercicio 2 - Control de Activos:** Procesamiento de inventario técnico mediante **matrices de NumPy**, permitiendo cálculos masivos de valorización.
4.  **Ejercicio 3 - Analítica de Sistemas:** Ejecución de funciones externas para el cálculo de métricas de clasificación (F1-Score) y disponibilidad de red.
5.  **Ejercicio 4 - Gestión CRUD POO:** Sistema de administración de actividades basado en **clases**, permitiendo operaciones de Crear, Leer, Actualizar y Eliminar (CRUD).
